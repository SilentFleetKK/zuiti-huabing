> **专治老板画饼：把"明年肯定优先考虑你"翻译成"什么条件、什么时候、怎么兑现"。**

老板留你开完会，一句话叠四张饼——成长机会、先扛下来、明年优先考虑你、年轻人别盯着眼前那点钱。听着很暖，一分钱不提。你被噎住，答应也不是，翻脸也不是。

画饼包是 [嘴替.skill](https://github.com/SilentFleetKK/zuiti-skill) 的画饼专场子仓。它只干一件事，但干得比谁都懂：**把老板嘴里模糊的未来，逼成纸上清楚的条款。**

给你三档能直接复制发出去的回复——接住活、钉住饼、饭碗和体面都还在。

## 看效果

### 明年优先考虑你

> **老板**（把你单独留下）
> 先把这摊事扛下来，明年调薪升职肯定优先考虑你。年轻人格局大一点，别老盯着眼前那点钱。

**嘴替·画饼包：**

🎭 **潜台词翻译**
用没时间、没书面、可反悔的"明年"，换你现在确定的多干和白干。"肯定""优先""考虑"三个词，每个都是给他留的退路。"别盯着眼前那点钱"是让你羞于开口。
**误读风险：低**。四种画饼话术叠着出现，还主动把补偿问题按下去，这是成套的成本转移，不是随口鼓劲。

💬 **三档回复**

【体面版】
谢谢领导信任，这摊事我接，也确实想借它练出来。关于明年调薪升职我挺期待的，方便的话咱定个大概的目标和时间点，我心里有个盼头，冲起来也更有劲。

【绵里藏针版】
没问题，活我扛。格局我是有的——也正因为想把这局做大，我想跟您把话说踏实一点：达到什么结果、大概什么时候兑现，咱简单对齐一下，邮件里同步一句就行。不是跟您计较眼前，是想冲得踏实、以后也不会有误会。

【掀桌版】
领导，活我全力以赴，这个没二话。但咱把话分开说：我出的力是现在、确定的；"明年优先考虑"是将来、您说了算的——这不对等。所以具体的条件和时间点，咱邮件里对齐一下。不是不信您，是白纸黑字双方都清楚。这才是真有格局。

📌 **发送建议**
优先发【绵里藏针版】：活接了，饼也钉上了时间和邮件，关系成本最低。

### 把公司当自己家

> **老板**
> 咱们是狼性团队，别有那么重的职业倦怠。把公司当成自己家，你现在多担一摊，等公司做大了，你就是元老。

**嘴替·画饼包：**

🎭 **潜台词翻译**
用"一家人"稀释雇佣关系，好让你像家人一样不计报酬地加班；用"元老"这个没定义的头衔，换你现在实打实多管一摊事。"等公司做大"是个永远在将来的条件。
**误读风险：低**。狼性 + 一家人 + 未来头衔三件套，指向的都是"现在多担、报酬再说"。

💬 **三档回复**

【绵里藏针版】
懂，我把这摊接过来。当自己家我没意见——不过家里也讲分工、也算账，才处得长久。这摊多出来的活和排期，咱在群里对齐一下归属；"做大了算元老"这话我记着，也想请您帮我把它落到一个大概的时间和标准上，我好安心冲。

📌 **发送建议**
"一家人"这类话别正面否定，否定就成了你无情。接住它，再顺势要分工和留痕——你接了活，也把"家人不算账"这个隐含前提退了回去。

## 安全边界

画饼包的"最狠档位"是把"拿未来换现在"讲透、把条件摆上桌、守住底线、不撕破脸。

- 不带脏字，不做人身攻击，不威胁，不造谣。
- 不教唆报复、逼宫、负气辞职或散布对公司的不实言论。
- 不替你决定该忍、该争还是该走，只把选项和后果摆清楚。
- 涉及签字、竞业、对赌、拿工资换期权时，优先提醒你看清合同、必要时找专业人士，不停在话术层面。
